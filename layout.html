Projects

my-secret-vault

Description: The "Personal Secret Vault" is a secure web application designed to help users store and manage their personal information in a private and organized manner. Upon logging in, users are presented with three main sections: Home, Dashboard, and Profile. The Home section allows users to pin important items for quick access. The Dashboard offers various tools, including a Personal Diary for daily entries, a section for uploading and organizing personal photos, a To-Do list manager, a My Targets section for setting and tracking goals, and My Docs for storing documents in PDF, Word, and Excel formats. Each feature is designed with security and user privacy in mind, allowing users to set additional passcodes for sensitive sections. The application ensures data integrity and security while providing a seamless and intuitive user experience.

Technologies Used: HTML, CSS, JAVASCRIPT, JAVA, REACT, SPRING, MY SQL, SPRING SECURITY

Smart Assistant using Python

Description: I designed and built a smart assistant using Python that can perform various tasks through both voice commands and text messages. The assistant is capable of browsing the internet, sending messages, operating the system, and more, all through natural language processing (NLP). The assistant also has a customizable wake word feature using Snowboy, allowing the user to activate it by saying a specific phrase or through a text message. I utilized various libraries, including SpeechRecognition, pyttsx3, pywhattkit, wikipedia, and Os, to enable these features. Additionally, I implemented a conversation management system using NLP techniques with the NLTK library, allowing the assistant to understand and respond to user queries in a more conversational and natural manner.
